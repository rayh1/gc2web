# ADR-0001: Warm archive visual system and typography hierarchy

Date: 2026-06-25
Status: Accepted

## Context

The visual refresh effort needs a cohesive identity across the homepage, entity pages, search, browse, and shared navigation. The existing site uses a cool blue/gray Tailwind palette and generic prose styling, which does not satisfy the PRD's requirement for a warm family-archive presentation.

## Decision

Adopt a shared semantic design-token layer with a warm palette, archive-oriented display typography for headings, and a legible sans-serif body/interface stack across all public page types and both theme modes.

## Consequences

- Visual changes can be applied consistently through shared layouts, components, and theme tokens.

* The redesign must retune both light and dark mode rather than only reskinning individual pages.
